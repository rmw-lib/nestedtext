# @rmw/nestedtext

## English Readme

use https://github.com/npillmayer/nestext wasm for nestedtext decode in nodejs ( es module )

###  Install

```yarn add @rmw/nestedtext ``` or ``` npm install @rmw/nestedtext```

### Use

```coffee
#include ./test/index.coffee
```

output is :

```json5
#include ./test.out.json5
```
